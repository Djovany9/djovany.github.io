titre : Mise en place d’un réseau virtuel d’entreprise avec un pare-feu pfSense
description : Dans ce projet, j’ai conçu et déployé une infrastructure réseau complète simulant un environnement d’entreprise. Le réseau virtuel comprend un pare-feu pfSense faisant office de routeur, un serveur Windows Server
Dans le cadre de mes compétences en administration réseau et système, j’ai réalisé la mise en place d’une infrastructure virtuelle d’entreprise comprenant un pare-feu pfSense assurant le routage et la sécurité du réseau, un serveur Windows Server configuré avec Active Directory, DNS et DHCP pour centraliser la gestion des utilisateurs et des ressources, ainsi que plusieurs postes clients reliés au domaine. J’ai conçu le plan d’adressage IP, configuré les rôles serveurs, établi les règles de filtrage réseau, géré les autorisations et l’authentification, et testé la communication entre les différentes machines virtuelles. Ce projet m’a permis d’appliquer des notions essentielles telles que la segmentation réseau, la sécurisation des accès, la gestion de domaine et la virtualisation, tout en acquérant une expérience concrète proche d’un contexte d’entreprise.
